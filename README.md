# Online Workshop Platform

A simple and clean web platform designed to manage online workshops, training events, and learning sessions.

## Features

###  Admin Panel
- Create workshops  
- Upload training materials  
- Add date & time  
- Provide descriptions  

###  User Dashboard
- View available workshops  
- Register for sessions  
- Access learning materials  

##  Technologies Used
- HTML5  
- CSS3  
- Responsive UI Design
